# Collapsible Component

A component system for creating expandable/collapsible sections controlled by a trigger element.

## Files

- `collapsible.svelte`: Main component that manages the open / closed state
- `collapsible-trigger.svelte`: Button or clickable element that toggles the collapsible state
- `collapsible-content.svelte`: Container for content that expands and collapses
- `index.ts`: Barrel export for easy importing all collapsible components
