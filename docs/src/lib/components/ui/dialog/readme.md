# Dialog Component

A modal dialog component system with trigger, overlay, content, and close button parts.

## Files

- `dialog-trigger.svelte`: Element that opens the dialog when clicked (typically a button)
- `dialog-overlay.svelte`: Semi-transparent backdrop overlay behind the dialog
- `dialog-content.svelte`: Main container for dialog content
- `dialog-close.svelte`: Button element for closing the dialog
- `index.ts`: Barrel export for easy importing all dialog components
