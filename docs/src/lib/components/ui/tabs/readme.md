# Tabs Component

A tabbed interface component system for switching between multiple content panels.

## Files

- `tabs.svelte`: Main component that manages tabs state and context
- `tabs-list.svelte`: Container for the tab trigger buttons
- `tabs-trigger.svelte`: Individual tab button that activates a content panel
- `tabs-content.svelte`: Content panel associated with a tab trigger
- `tabs.css`: Styling for the tabs layout and transitions
- `index.ts`: Barrel export for easy importing all tabs components
