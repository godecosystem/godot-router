# Card Component

A multi-part component system for building card layouts with header, content, footer, and meta sections.

## Files

- `card.svelte`: Main card container component that wraps the entire card
- `card-header.svelte`: Header section composition (typically contains title and description)
- `card-title.svelte`: Title text for the card header
- `card-description.svelte`: Descriptive text for the card header
- `card-content.svelte`: Main content area of the card
- `card-footer.svelte`: Footer section (typically contains action buttons)
- `index.ts`: Barrel export for easy importing all card components
