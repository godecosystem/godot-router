# Code Group Component

A tabbed component system for displaying code examples with multiple language/variant options.

## Files

- `code-group.svelte`: Main component that manages tabs and content switching
- `code-group-list.svelte`: Container for tab trigger buttons
- `code-group-trigger.svelte`: Individual tab button for selecting code variants
- `code-group-content.svelte`: Container for the code block content displayed for each tab
- `code-group.css`: Styling for the code group tabs and layout
- `index.ts`: Barrel export for easy importing all code-group components
