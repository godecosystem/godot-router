# Dropdown Menu Component

A dropdown menu component system with trigger, content, and individual menu items.

## Files

- `dropdown-menu-trigger.svelte`: Button or element that opens the dropdown menu
- `dropdown-menu-content.svelte`: Container for the menu items that appears below / beside the trigger
- `dropdown-menu-item.svelte`: Individual selectable menu item
- `index.ts`: Barrel export for easy importing all dropdown-menu components
