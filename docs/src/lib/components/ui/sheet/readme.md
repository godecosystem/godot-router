# Sheet Component

A sliding panel/drawer component system that typically appears from the edge of the screen.

## Files

- `sheet-trigger.svelte`: Element that opens / toggles the sheet panel
- `sheet-overlay.svelte`: Semi-transparent backdrop overlay behind the sheet
- `sheet-content.svelte`: Main container for the sheet content
- `sheet-close.svelte`: Button element for closing the sheet
- `index.ts`: Barrel export for easy importing all sheet components
